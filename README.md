# BeautyCalendar

This project lets the user play the "mobile flashcards" on the mobile phone: create new tests and test his/her knowledge

## How to install the project

- clone the repository
- cd to the main directory
- npm install
- npm start

## Tested environments

- iOS simulator (xcode)
